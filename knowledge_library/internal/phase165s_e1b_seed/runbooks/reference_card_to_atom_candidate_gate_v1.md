# Reference Card → Atom Candidate Gate v1

A reference card is not accepted memory.

Atom candidate is allowed only when all are true:
1. There was a real task, parent task, owner task, autonomous step, or school lesson.
2. Builder hit a concrete gap.
3. Builder searched accepted memory/internal reference/governed external source.
4. Builder found an answer.
5. Builder applied the answer to the task.
6. There is proof/use that the answer worked.
7. The knowledge is likely reusable.
8. The candidate is bounded and can pass C2B/PHASE162.

If not true:
store as reference/material candidate only.
