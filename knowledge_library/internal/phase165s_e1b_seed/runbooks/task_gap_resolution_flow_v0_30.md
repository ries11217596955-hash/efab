# Task Gap Resolution Flow V0.30

```text
task arrives
→ decompose task
→ detect missing capability/knowledge
→ scan accepted atoms/molecules/organs
→ scan internal reference library
→ scan self-map/body-map/control files if relevant
→ if still missing: governed external search policy
→ apply found answer
→ produce proof/use
→ if reusable and bounded: atom candidate via C2B/PHASE162
→ if no answer: record source_gap and escalate/return to parent flow
```

## Source gap record minimum
- gap_id
- task_context
- missing_knowledge
- internal_sources_checked
- external_allowed
- proposed_reference_card
- proposed_material_candidate
- risk
- next_action
