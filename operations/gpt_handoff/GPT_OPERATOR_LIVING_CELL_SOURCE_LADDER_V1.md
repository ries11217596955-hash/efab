## Growth hierarchy

Use the hierarchy below for Builder construction:

```text
signal / symbol
-> atom
-> molecule
-> tissue
-> organ
-> circuit
-> system
-> organism
-> child-agent production
```

Definitions:

- Signal/symbol: raw A, B, C, D-level element; not useful alone.
- Atom: smallest accepted rule/fact/action unit with proof boundary.
- Molecule: stable grouping of atoms that creates a reusable concept, e.g. A+B+C+D -> alphabet.
- Tissue: repeated molecules that serve one function class.
- Organ: authority-bound capability with input, process, output, validator, rollback.
- Circuit: organs connected by contracts.
- System: circuits forming a body function.
- Organism: self-observing, self-changing, self-verifying Builder.
- Child-agent production: organism creates bounded child agents after self-build is mature.
