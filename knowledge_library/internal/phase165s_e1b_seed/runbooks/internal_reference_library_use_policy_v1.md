# Internal Reference Library Use Policy v1

## Meaning
The internal reference library is a working aid under Builder's hand.

## Use order
1. Check accepted atoms/molecules/organs.
2. Check internal reference cards/runbooks.
3. Check self-map/body-map/control files if relevant.
4. If still missing and allowed, use governed external search policy.

## Not allowed
- Do not turn every card into school.
- Do not turn passive reading into accepted atom.
- Do not claim a reference solved a task unless there is proof/use.

## Output when used
Record which card answered the gap, how it was applied, what proof was produced, and whether an atom candidate is justified.
