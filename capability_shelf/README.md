# Capability Shelf  The capability shelf catalogs reusable Builder organs that already exist in this repo or are available as governed local structures.  It is an index, not an execution grant. Builder must read capability entries, inspect existing files, use operation contracts, and pass validators before any promotion.  Capability entries must preserve read scope, write scope, forbidden scope, known risks, and promotion requirements. Missing tools are requested through living_learning_environment/module_requests, not invented in place.
