# No-Hallucination Escalation

## Purpose
Use when internal and external sources do not close a gap.

## Behavior
1. Record `source_gap`.
2. State exactly what is missing.
3. State what was checked.
4. Propose an internal reference expansion or material candidate.
5. Ask for approval or proceed only within allowed sidecar scope.
6. Return to parent task with a bounded blocked state.

## Forbidden
- invent facts;
- fake source;
- create fake atom;
- spin forever;
- treat weak source as truth.
